discord_bot_學習心路歷程

想不開的自虐計畫

一次突然的想法，突然想去觸碰自己從來都沒有觸碰過的領域，回復式機器人，當初想說:網路上的資源那麼多，應該學起來很容易吧。就因為這種想法讓我踏上了這痛苦的不歸路

一開始的學習規劃

選擇程式語言

在一開始我上網爬資料時我看到了編寫discord bot的程式語言主要有兩種

  • JavaScript
  • Python
    最露我決定了從學習起來簡單許多的Python，當初的我還沒意識到的這是一個錯誤的決定

尋找學習資源

因為之前我就學簡單Python基礎，所以我著手先去Youtube上尋找discord-bot的教學和上discord官方網站去學習discord.py

  • 學習影片: https://www.youtube.com/watch?v=8sQAgaaEl0M

  • Discord.py: https://discordpy.readthedocs.io/en/stable/api.html

立定製作目標

  • 簡單的文字回復
  • 圖片回復
  • 定時推送
  • 身分組派發
  • 音樂機器人

著手進行

我開始一步步的按照影片慢慢地學習，一開始事情蠻順利的也沒有碰到什麼太大的困難，也稍微完成了一些指令

  • 簡單的文字回復

  • 匿名說話

碰到困難

就當我想說可以安穩的寫完的時候，我遇到了困難discord.py更新(discord政策更新)。這對當初還不太熟悉的我是一個重大問題，一開始我也和其他人一樣慢慢修改程式，但還是因為不夠熟悉而無法解決問題，也因此休息了一段時間
